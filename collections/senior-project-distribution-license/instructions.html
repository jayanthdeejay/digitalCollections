Instructions - Senior Student Projects

INSTRUCTIONS

Please attach and submit your final report in the BlackBoard page for your senior project class.

File format guidelines: Submit senior design reports as a single PDF file consisting of the report and any supplementary files (PPT slides, CAD drawings, etc.) Proprietary formats are strongly discouraged (e.g. Microsoft Office documents). If you have a video or other file that is unable to be placed into a PDF format, please contact Ted Baldwin (Ted.Baldwin@uc.edu) at the CEAS Library for assistance. The full version of Adobe Acrobat (PDF editing software) is available on four computers in the CEAS Library (850 Baldwin Hall).
